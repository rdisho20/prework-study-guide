CSS

  • The `*` selector applies styles globally (e.g., `margin: 0; padding: 0;`).
  • Multiple selectors can share styles using commas (e.g., `header, footer`).
  • DRY (Do Not Repeat Yourself) encourages reusing code for efficiency.
  • Text can be centered with `text-align: center` for `h1` and `h2`.
  • Images can use `display: block` and `margin: auto` for centering.
  • `ul` lists can have custom padding (`padding-left: 40px`) and font sizes.
  • Paragraphs (`p`) can be centered with a font size of `30px`.

"The" Box Model

CSS Box Model: Defines how elements render, encompassing content, padding, border, and margin.

Key Concepts:

  1. Box Types:
    • Block: Starts on new line, respects dimensions, affects spacing w/ padding, borders, and margins.
    • Inline: Flows in-line, ignores height/width, affects horizontal spacing.
  2. Box Layers:
    • Content Box: For content (`width`, `height`).
    • Padding Box: Space around content (`padding`).
    • Border Box: Around padding (`border`).
    • Margin Box: Space between elements (`margin`).
